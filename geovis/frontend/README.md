# Frontend Application

## Overview
This is the React client for the GeoVis project, built with Vite. It visualizes restaurant data using deck.gl, MapLibre GL, and Plotly.

## Installation

1.  Navigate to the frontend directory:
    ```bash
    cd geovis/frontend/geovis
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Running the App

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
